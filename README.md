# Notes Wall Web Application

A simple web application for taking and managing notes.

## Features

- Add notes with custom colors.
- Delete individual notes.

## Technologies Used

- HTML
- CSS (Bootstrap)
- JavaScript

## Getting Started

1. Clone this repository to your local machine.
2. Open the `index.html` file in your web browser.

## Usage

1. Enter your note in the textarea.
2. Choose a color for the note.
3. Click the "Add note" button to add the note to the wall.
4. To delete a note, click the "Del" link on the note.
